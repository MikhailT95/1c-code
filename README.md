# 1c-code

Обезличенный пример псевдокода на встроенном языке 1С.

В репозитории:

- `src/integration-example/ManagedForm.bsl` — пример модуля управляемой формы
- `src/integration-example/IntegrationServer.bsl` — пример серверного общего модуля
- `src/calendar-organizer-example/ManagedForm.bsl` — пример формы календарного органайзера
- `src/calendar-organizer-example/CalendarOrganizerServer.bsl` — пример серверной логики календаря

Что показано в шаблоне:

- разделение формы и серверной бизнес-логики
- загрузка данных из внешнего REST API
- формирование документов по загруженным данным
- построение календарного органайзера по статусам и периодам
- безопасное хранение токена через безопасное хранилище
- базовая обработка ошибок и запись в журнал регистрации

Это именно демонстрационный шаблон, а не готовый код под конкретную базу.
